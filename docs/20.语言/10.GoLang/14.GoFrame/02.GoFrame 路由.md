## 配置文件

``` yaml
server:
    ...
    routeOverWite: true
    ...
```

## 重写路由

``` go
	s := g.Server()
	s.Group(consts.BackgroundUrl, func(group *ghttp.RouterGroup) {
		group.Middleware(adminMiddleware.Middleware().BaseGeneral)
		group.Bind(admin.AdminIndex)
	})
```

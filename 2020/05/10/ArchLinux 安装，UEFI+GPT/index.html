ArchLinux 安装，UEFI+GPT

安装前

设置键盘布局

可以省略，默认为us键盘布局

  • 查看可用的键盘布局
  • 设置键盘布局
验证启动方式

以下命令执行成功则为uefi模式启动

  • 执行命令
连接网络

最好使用有线连接，因为简单

  • 测试网络是否已经连接
  1. 使用有线连接网络
  1. 使用无线连接

使用wifi-menu命令，命令可能无法使用，需要先安装wpa_supplicant和dialog模块

安装wpa_supplicant

更新系统时间
  • 设置时间同步

注意用date看一下时间，与正确时间差了8个小时

对磁盘进行分区

目前划分的是3个分区，/boot，/home,/

  • 对磁盘进行分区

    • 使用parted进行磁盘分区

      • 查看磁盘设备

      以下为虚拟机上的分区，分区表格式为mbr

      • 选择操作设备块
      • 修改为GPT格式
      • 删除已有分区

      使用rm 数字删除分区

      • 新建分区

        • 新建boot分区
        • 新建home分区
        • 新建根分区
        • 查看分区信息

      • 退出(parted) quit

    • 使用fdisk进行磁盘分区

      进行fdisk的命令交互操作时候不能删除输入的内容

      • 选择操作设备块

      • 创建一个empty gpt分区表

      • 创建分区，作为boot分区

      • 创建分区，作为home分区

      • 创建分区，作为/分区

      • 查看分区信息

      • 格式化boot分区

        必须为fat32格式

      • 格式化home,/分区

挂载分区

  • 挂载/分区

  • 挂载boot,home分区

    一定要先挂载/分区，不然这两个挂载点会被覆盖

安装

选择镜像源地址

这一步可以跳过，安装过程中网速可能有些慢

  • 安装vim

    vim最好装上

  • 修改镜像源地址

安装基础包

也可以使用# pacstrap /mnt base base-devel

配置系统

生成挂载文件（开机自动挂载分区）

生成之后检查一下文件内容，应该有3个分区被挂载

切换到新系统中

设置本地时区

  • 设置时区

    再看时间是对的

  • # ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

# hwclock --systohc

# locale-gen

# vim /etc/locale.conf

设置主机名

将/etc/hostname文件的内容修改为archlinux

修改hosts文件

添加以下内容到/etc/hosts

127.0.0.1 localhost
::1 localhost

网络配置

  • 安装dialog, wpa_supplicant

    安装wpa_supplicant后可以使用wifi-menu命令，用界面连接wifi，依赖dialog模块
